Very basic local html scraper






This program is for scraping information from local HTML files. It is a quick implementation and is very buggy.

  1. Find a page you want to scrape.
  2. Copy the html to a file that contains the basic html structure base (html,head,body)
  3. Save this file into scraper/data/files/
  4. Bring up Vagrant and run server with node index.js
  5. Open localhost:3000/index.html
  6. Give scraper the name of the file you want to scrape, the tags you want to use and type for the file you want to download
Query is done with Cheerio and it uses jQuery so googling solutions for that is useful. For example choosing text for only childless elements input: p:not(:has(>div))
